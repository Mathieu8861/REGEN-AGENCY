4.5Nom de domaine vs Hébergement

⚠️ IMPORTANT - À ne pas confondre :
Le nom de domaine (OVH, Ionos...) et l'hébergement (Vercel, Railway...) sont deux choses distinctes.
SCHÉMA EXPLICATIF
NOM DE DOMAINE
OVH, Ionos, Gandi...
  • C'est l'adresse web
  • exemple.fr
  • Coût : ~10€/an
  • Seul achat chez OVH/Ionos
~10€/AN
≠
HÉBERGEMENT
Vercel, Railway...
  • C'est le serveur
  • Stocke les fichiers du site
  • Vercel = GRATUIT
  • Pas besoin d'OVH/Ionos
GRATUIT
Ce qu'on achète Où ? Prix Pourquoi ?
Nom de domaine OVH, Ionos, Gandi ~10€/an Pour avoir une adresse personnalisée (exemple.fr)
Hébergement Vercel GRATUIT Pour stocker et servir les fichiers du site
En résumé pour un site vitrine :
  • ✅ Acheter uniquement le nom de domaine chez OVH/Ionos (~10€/an)
  • ✅ L'hébergement est 100% gratuit sur Vercel
  • ❌ Pas besoin d'hébergement payant chez OVH/Ionos

→ Coût total annuel d'un site vitrine : ~10€/an (domaine seul)

8Conformité RGPD / Union Européenne

✓ Hébergement 100% EU confirmé
Toutes les données sont hébergées dans l'Union Européenne, en conformité avec le RGPD.
Service Région EU Localisation Conformité
Vercel ✓ fra1 Frankfurt, Allemagne RGPD, ISO 27001
Railway ✓ eu-west Europe de l'Ouest RGPD, SOC 2
Supabase ✓ eu-central-1 Frankfurt, Allemagne RGPD, SOC 2, HIPAA
Stripe ✓ EU Dublin, Irlande PCI DSS, RGPD

Mesures RGPD implémentées

Localisation des données Toutes les bases de données hébergées sur serveurs EU (Frankfurt)
Pas de transfert US Aucune donnée personnelle transférée hors UE
Chiffrement Données chiffrées au repos et en transit (TLS 1.3)
Droit à l'oubli Suppression des données clients sur demande via Supabase
Portabilité Export des données au format standard (JSON, CSV)
Consentement cookies Consent Mode V2 implémenté sur tous les sites
DPA (Data Processing Agreement) Contrats de sous-traitance signés avec chaque fournisseur
Configuration requise à la création de chaque projet :
  • Vercel : Sélectionner "Frankfurt (fra1)" dans Project Settings → Functions Region
  • Railway : Sélectionner "EU West" lors de la création du projet
  • Supabase : Sélectionner "Frankfurt (eu-central-1)" lors de la création du projet
